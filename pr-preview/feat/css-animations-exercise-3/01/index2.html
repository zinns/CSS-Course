Resumen de html

html es una herramienta de marcado de hiper texto que se usa para diseñar paginas webs, en ella podemos poner textos, agregar botones imagenes , videos , links y hacer tablas.

html nos ayuda a ir creadon una estructura basica de como se veria una pagina , podemos poner links para compartir informacion , musicas o videos de otras paginas e igualmente podemos colocar pequeñas o grandes secciones de otras paginas en la nuestra con un iframe el cual uno de los usos mas frecuentes es poner una ubicacion de google maps.

hacer un pagina web puede llegar hacer algo enredado en algunas ocaciones, por lo cual es bueno tener algunas estructuras que ayuden a mantener un orden en el codigo.

Los contenedores son una forma de tener un orden el codigo , junto a trabajar con el html semantico es una de las formas de tener un orden el la pagina y en el codigo para poder tener un manejo mas facil en caso de tener que modificar algun aspecto de la pagina.

html es una estructura en la que con otras herramienta como css se les puede poner color , fuentes a el texto y estilos.
