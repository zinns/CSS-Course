Resumen de introduccion a Css

En la introduccion a css se vieron varios temas como los NPM que significan node package manager que son librerias para usar en un proyecto como animaciones , ventanas emergentes y algunos efectos ya que son cosas que ya estan hechas en una libreria y su uso llega a ser mas practico ya que no tienes que hacerlas desde cero.

Tambien vimos otras librerias como yarn, pnpm, bun las cuales igual son practicas para el uso en un proyecto de desarollo.

Otra cosa importante que vimos en la introduccion de css fue el package json que son requerimientos para la apliccion o para el desarollador que ayudan a que todo sea mas facil y compatible con tu area de trabajo.
