Resumen de introduccion a Css

En la introduccion de CSS vimos que es y para que sirve. Y lo primero es que significa "CSS" y significa "Cascading style sheet" y español seria "hojas de estilo en cascada" y la principal funcion es dar un estilo a la pagina con color , poner diferentes fuentes a las letras y cambiar sus tamaño, degradados y hasta animaciones.

Estos cambios de aparencia se pueden aplicar tanto a letras como agrupos o secciones de los contenedores y de esta forma modificar tambien el tamaño del contenedor en si usando una propiedad y cambiando el valor. Ya sea que cambies el color con RGB, RGB Alphay Exadecimal.

Tambien vimos otras librerias como yarn, pnpm, bun las cuales igual son practicas para el uso en un proyecto de desarollo.

Otra cosa importante que vimos en la introduccion de css fue el package json que son requerimientos para la apliccion o para el desarollador que ayudan a que todo sea mas facil y compatible con tu area de trabajo.
